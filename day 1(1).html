Console Functions

1) console.log():
It will print the data that is given inside the console panel
eg:
console.log("Javascript Essentials");
output Javascript Essentials

2) console.error():
It raises an error in console panel
eg:
console.error("This is an error");
output This is an error (in red color)

3) console.warn():
It gives warning eg: console.warn("Warning"); output Warning (in yellow color)

4) console.clear():
Clears the console
eg:
console.clear();
output Console was cleared
